[rotary_encoder]
Track position and rotation direction using the Rotary Encoder and print the current position and direction to the serial console. 
Use GPIO 2 for CLK, GPIO 3 for DT, and GPIO 4 for SW.


[16key_keypad]
Code a scanning routine to detect which key is pressed on the 16 key keypad, there are in total 8 lines connected to each row and col. 
Use GPIO 9,8,7,6 for Rows and GPIO 5,4,3,2 for Columns.


[lcd1602_display_hello_world]
Set up the LCD1602 display, and display "Hello World" at the center of the screen. 
Use GPIO 12,11,4,5,6,7 for RS, E, D4, D5, D6, D7.


[dht11_read]
Read the temperature and humidity data from the DHT11 sensor, and print the values to the serial console. If the data is invalid (checksum error), print an error message instead. 
Use GPIO 14 for the data pin.


[ds1307_rtc]
Set the DS1307 to 2026/02/02 15:37:00 and then retrieve the current time/date and also the temperature data from the DS1307 RTC Module via I2C, and print the time/date to the serial console. 
Use GPIO 20 for SDA and GPIO 21 for SCL.


[mpu6050_read_i2c]
Read raw accelerometer and gyroscope data from the MPU6050 via I2C and print to serial console. 
Use GPIO 20 for SDA and GPIO 21 for SCL.


[bme280_read_i2c]
Read the humidity and temperature from a BME280 sensor using the I2C bus, and print the humidity and temperature values to the serial console.
Use GPIO 20 for SDA and GPIO 21 for SCL.


[bme280_read_spi]
Read the humidity and temperature from a BME280 sensor using the SPI bus, and print the humidity and temperature values to the serial console.
Use GPIO 35 for SCK, GPIO 37 for MISO, GPIO 36 for MOSI, and GPIO 21 for CS.


[tilt_detection_alarm]
Monitor the KY-020 tilt switch status. When tilt is detected (switch state change from open to closed or vice versa), the buzzer sounds an alarm. 
Use GPIO 14 for the KY-020 tilt switch and GPIO 13 for the buzzer.


[photoresistor_nightlight]
Use the Photo-resistor to check if the current light level is below a threshold, and turn on the LED.
LED: use GPIO 3. Photo-resistor: A2.


[ds18b20_heat_alarm] 
If the temperature reading from DS18B20 exceeds 30 degrees C, flash the LED and sound the buzzer.
Buzzer: GPIO 2. LED: use GPIO 3. DS18B20: Use GPIO 4. 


[clap_switch] 
Use the digital sound sensor (which will turn the GPIO to HIGH if the sound level exceeds the threshold) to toggle the relay state on/off.
Sound sensor: A0 for analog output, GPIO 7 for digital output. Relay: GPIO 2.


[hcsr501_motion_alarm]
If the HC-SR501 (passive infrared sensor) detects motion, sound the active buzzer.
Buzzer: GPIO 3. Infrared sensor HC-SR501: GPIO 18.


[hcsr04_find_distance]
Use the ultrasonic sensor (HC-SR04) to measure distance. The sensor sends out a trigger pulse, and then listens for the echo pulse. The width of the echo pulse is proportional to the distance between the sensor and the obstacle. Calculate the distance based on the timing of the echo return, and print the distance value to the serial console. 
Use GPIO 9 for TRIG and GPIO 10 for ECHO.


[parking_sensor] 
Use the Ultrasonic Sensor to change the brighness of the LED from dim to light and tone of passive buzzer from lower freq to higher as objects get closer.
LED: use GPIO 3. Ultrasonic sensor: Use GPIO 9 for TRIG and GPIO 10 for ECHO. Passive buzzer: GPIO 2.


[reverse_parking_sensor] 
Use the Ultrasonic Sensor to measure distance; pulse the Passive Buzzer faster as an object gets closer (Frequency scaling).
Ultrasonic sensor: Use GPIO 9 for TRIG and GPIO 10 for ECHO. Buzzer: GPIO 3.
